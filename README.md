# 🎯 运筹学优化算法学习工具包

## 🚀 快速开始

```bash

```

## 🖼️ 生成图片详解

### 📈 如何分析和理解这些图表

#### 1. `feasible_region.png` - 可行域图

**看什么：**

- 蓝色区域：所有约束条件都满足的解的集合
- 红色线：约束边界
- 红点：最优解位置

**学会什么：**

- 理解约束如何限制解的范围
- 最优解总是在可行域的顶点
- 约束越多，可行域越小

#### 2. `linear_programming_results.png` - 线性规划结果

**看什么：**

- 柱状图：各产品的最优生产量
- 饼图：资源利用率
- 表格：详细的数值结果

**学会什么：**

- 如何解读最优解
- 资源是否充分利用
- 哪些约束是紧约束（起决定作用）

#### 3. `optimization_process.png` - 优化过程动画

**看什么：**

- 红色轨迹：算法搜索路径
- 等高线：目标函数值相等的点
- 箭头：梯度方向

**学会什么：**

- 优化算法如何逐步找到最优解
- 梯度下降的工作原理
- 收敛过程

#### 4. `3d_optimization.png` - 三维优化表面

**看什么：**

- 彩色表面：目标函数的三维形状
- 最低点：全局最优解
- 等高线：函数值相等的曲线

**学会什么：**

- 多变量优化问题的复杂性
- 局部最优与全局最优的区别
- 约束对解空间的影响

#### 5. `network_flow.png` - 网络流图

**看什么：**

- 节点：网络中的关键点
- 边：连接和容量
- 颜色：流量大小

**学会什么：**

- 网络结构对流量的影响
- 瓶颈在哪里
- 如何增加网络容量

#### 6. `sensitivity_analysis.png` - 敏感性分析

**看什么：**

- 曲线变化：参数改变对最优解的影响
- 斜率：敏感程度
- 拐点：临界值

**学会什么：**

- 哪些参数对结果影响最大
- 解的稳定性
- 决策的风险评估

### 📊 学习建议

1. **先看图，再看数字**

   - 每次运行后，先观察生成的图片
   - 理解图表含义后，再分析控制台输出的数值
2. **对比不同算法**

   - 运行多个演示，比较它们解决问题的方式
   - 注意各算法的适用场景
3. **修改参数实验**

   - 尝试修改代码中的参数值
   - 观察结果如何变化，加深理解
4. **关联实际问题**

   - 思考这些算法如何应用到你熟悉的领域
   - 尝试用学到的方法解决实际问题
