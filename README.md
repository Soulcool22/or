# 运筹学优化演示项目

## 📋 项目概述

本项目提供了全面的运筹学优化算法演示，包括理论计算、实际应用和可视化展示。通过Python实现多种经典优化算法，帮助理解运筹学在实际问题中的应用。

- **整数规划** - 设施选址、项目选择

- **网络流** - 最大流、最小费用流
- **车辆路径问题 (VRP)** - 配送路径优化

## 🏗️ 项目架构

### 模块化设计
本项目采用模块化架构，将原始的单一文件拆分为多个专门的演示模块：

- **🎯 main_demo.py**: 主程序入口，提供交互式菜单和整合功能
- **📚 演示模块**: 每个优化问题类型独立成模块，便于学习和维护
- **🔧 原始文件**: 保留原始演示文件，确保向后兼容性

### 模块特点
- **独立性**: 每个模块可以单独运行，不依赖其他模块
- **完整性**: 每个模块包含完整的问题求解、可视化和报告功能
- **一致性**: 所有模块采用统一的代码结构和接口设计
- **扩展性**: 易于添加新的优化算法和问题类型

## 📊 演示内容详解

### 1. 线性规划 (Linear Programming)

**问题描述：** 多产品生产计划
- **目标：** 最大化利润
- **约束：** 资源容量限制
- **规模：** 50种产品，20种资源
- **算法：** 单纯形法 (通过PuLP调用)

**数学模型：**
```
maximize:   Σ(profit_i × x_i)
subject to: Σ(resource_matrix[i,j] × x_i) ≤ capacity_j  ∀j
            x_i ≥ 0  ∀i
```

### 2. 整数规划 (Integer Programming)

**问题描述：** 设施选址问题
- **目标：** 最小化总成本
- **约束：** 需求满足、设施容量
- **特点：** 0-1决策变量
- **算法：** 分支定界法

### 3. 运输问题 (Transportation Problem)

**问题描述：** 全国物流网络优化
- **目标：** 最小化运输成本
- **约束：** 供应量、需求量平衡
- **规模：** 15个供应商，25个客户
- **特点：** 特殊结构的线性规划

### 4. 网络流 (Network Flow)

**问题描述：** 最大流问题
- **目标：** 最大化从源到汇的流量
- **约束：** 弧容量、流量守恒
- **算法：** Ford-Fulkerson算法
- **应用：** 通信网络、交通流量

### 5. 车辆路径问题 (VRP)

**问题描述：** 配送路径优化
- **目标：** 最小化总行驶距离
- **约束：** 车辆容量、客户需求
- **算法：** 贪心启发式算法
- **扩展：** 时间窗、多车型

### 6. 投资组合优化 (Portfolio Optimization)

**问题描述：** 金融投资组合
- **目标：** 最小化风险（给定收益）
- **约束：** 预算、权重限制
- **模型：** 马科维茨均值-方差模型
- **指标：** 夏普比率、风险分散

## 🔍 算法性能分析

### 时间复杂度对比
| 算法类型 | 时间复杂度 | 适用规模 | 最优性保证 |
|---------|-----------|---------|-----------|
| 线性规划 | O(n³) | 大规模 | 全局最优 |
| 整数规划 | O(2ⁿ) | 中小规模 | 全局最优 |
| 网络流 | O(V²E) | 大规模 | 全局最优 |
| 启发式算法 | O(n²) | 超大规模 | 近似最优 |

### 求解器对比
- **PuLP**: 开源，易用，适合教学
- **Gurobi**: 商业，高效，适合工业应用
- **CPLEX**: 商业，功能全面
- **OR-Tools**: Google开源，综合性强
